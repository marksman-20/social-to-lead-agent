# Primetrade.ai Data Science Intern Assignment

This repository/folder contains the submission for the "Trader Performance vs Market Sentiment" assignment for the Data Science Intern role at Primetrade.ai.

## Contents
- **`assignment.py`**: The complete Python script doing the data processing, Exploratory Data Analysis, segmentation, and machine learning prediction.
- **`assignment.ipynb`**: The Jupyter Notebook equivalent of the python script containing identical executable code.
- **`report.md`**: A short max 1-page write up highlighting the methodology, key insights, and 2 actionable strategy outputs.
- **`*.png`**: Various visual summary charts generated by the script (`pnl_by_sentiment.png`, `freq_by_sentiment.png`, `segment_pnl.png`, `feature_importance.png`).
- **`data/`**: Directory containing the `sentiment.csv` and `trades.csv` datasets.

## Requirements
Please ensure you have Python 3.8+ installed. You also need the following standard data analysis packages:
```bash
pip install pandas numpy matplotlib seaborn scikit-learn jupyter
```

## How to Run
#### Option 1: Running the Jupyter Notebook
1. Open the notebook using Jupyter.
```bash
jupyter notebook assignment.ipynb
```
2. Run all cells sequentially.

#### Option 2: Running the Python Script
You can simply execute the script via the command line to recreate all metrics and visualizations from scratch.
```bash
python3 assignment.py
```
This will recalculate all statistics, output table summaries to standard output, and re-generate the PNG charting visuals locally.

## Evaluation Areas Covered
- **Data clean/merge**: Done efficiently converting IST times to cleanly map against GMT-based sentiment indices.
- **Reasoning**: Outlined carefully in the Markdown report addressing the nuance between High-frequent / Infrequent traders.
- **Actionable Execution**: Outlines 2 distinctive and measurable logic adjustments for trading algorithm allocations based on the behavioral traits mapped against Fear/Greed.
- **Bonus Predictivity**: Uses a lightweight Random Forest model to measure predictability of next-day trader success solely based on recent trait behaviors and sentiment values.
